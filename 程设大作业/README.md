

## 运行

进入目录：

启动图形界面：

```powershell
python main.py
```

## 已实现的主流程

1. 图形启动菜单：同屏、建房、加入、退出，并可切换 English / 中文。
2. 地图配置界面：三档地图大小、滑条自定义、环路复杂度、地图重合率、雾等级、地图特色等级、玩家数、碰撞和拖尾开关；同屏模式固定双人。
3. 角色选择界面：为每个玩家选择临时角色预设，并显示技能与冷却。
4. 同屏对战：P1/P2 分屏，各看自己的固定全图迷宫，视野范围跟随角色。
5. 雾等级：重雾、轻雾、无雾。
6. 任务系统：果实作为默认必做任务；轨迹转弯、闭合环路、不同环路、诱导陷阱、相遇作为随机奖励任务。
7. 技能/效果系统：视野、终点、地图修改、buff/debuff、传送、偷取等。
8. HUD：时间、任务进度、技能冷却、buff/debuff、事件播报、所有玩家百分比进度榜。
9. 终点默认锁定：收集指定数量真实果实后自动显示并允许进入，到达后获胜。
10. 联网大厅：客户端可选择角色，所有玩家准备后开始，Host 权威同步，断线席位可重新加入。

## 操作

菜单：

```text
鼠标点击按钮
1/2/3/Q 可在启动页快捷选择
E/C 可在启动页切换英文/中文
Enter 可在配置/角色页继续
ESC 返回或退出
```

同屏移动：

```text
P1：WASD
P2：方向键
```

技能：

```text
P1：1/2/3/4/5/6/7/8
P2：, . / ; [ ] - =
```

开局后操作：

```text
同屏模式：游戏开始后鼠标禁用，光标会隐藏。
同屏模式：所有技能按快捷键后立即释放。
同屏模式：格子类技能会自动选择角色面前，或当前锁定对手面前的格子/区域。
联网模式：格子类技能按快捷键后进入选点状态，再用鼠标左键点击释放位置。
联网模式：非格子类技能按快捷键立即释放；右键可在可见格子上做标记。
Tab 切换当前锁定的对手目标，debuff 和对方地图技能会作用于该目标。
M/N 可在当前位置放置/取消标记。
```

## Debug 检查

可以运行：

```powershell
python debug_checks.py
```

它会检查同屏技能默认目标、联网鼠标选点、同屏模式是否强制双人、以及启动/配置/角色/加入页面按钮是否互相遮挡。

## 联网试玩

同一台电脑测试：

1. 第一个窗口运行 `python main.py`，选择 `Host Network Room`，配置后开始。
2. 第二个窗口运行 `python main.py`，选择 `Join Network Room`，IP 输入 `127.0.0.1`。

两台电脑局域网：

1. 主机选择 `Host Network Room`。
2. 客户端选择 `Join Network Room`，输入主机电脑的局域网 IPv4 地址。

当前联网版本：主机控制 P1，客户端按加入顺序控制 P2/P3/P4；客户端加入前可选角色；所有配置人数都连接并准备后自动开始；断线后对应席位会空出，重新加入可复用空席位。
